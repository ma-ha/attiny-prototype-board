# ATtiny Prototype Board v2

Simplifies circuit prototypying with the ATtiny xx14.

![1stBuild](board-finished.png) 

Features:
- Beadboard (400 pins)
- I/O
  - 4x LED
  - 3x Potentiometer
  - 2x momentary push button
  - 4x Switch
  - 2x 3.5 mm TRS socket
  - 4x banana socket
  - 2/3x I2C
  - 2x Serial
- ATtiny XX14
- FTDI adapter
  - UPDI programming
  - USB power
  - Serial debugging
